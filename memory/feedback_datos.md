---
name: Feedback — manejo de datos y números
description: Cómo guardar y usar datos de negocio correctamente
type: feedback
---

No guardar snapshots de números — guardar cómo obtenerlos (queries, links a Sheets).

**Why:** Los números de MeLi cambian cada semana. Lo que sirve a largo plazo es la metodología: qué query usar, qué filtros aplicar, desde qué tabla o dashboard.

**How to apply:** Si el usuario comparte un número sin contexto (fecha, fuente), preguntar la fecha y si es dato actual o histórico. Priorizar guardar links a Sheets/dashboards y queries sobre valores puntuales.
